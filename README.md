# BackRoom with Memorization

이 게임은 색깔을 암기하여 아이템을 모으는 단순한 게임입니다. 게임 시작 시 3초 동안 "색깔 목록"이 표시되며, 이 순서대로 아이템을 모으면 게임이 종료됩니다. 적이 플레이어를 쫓아올 수 있으므로 아이템을 모으는 동안 조심해야 합니다.

## 게임 조작법

- w: 위로 이동
- a: 왼쪽으로 이동
- s: 아래로 이동
- d: 오른쪽으로 이동
- SHIFT: 전력 질주 (대쉬)
- ctrl: 앉기
- 마우스 오른쪽 버튼: 확대

## 설치 및 실행

1. 이 저장소를 클론합니다.

2. 게임 실행에 필요한 종속성을 설치합니다.

3. 게임을 시작합니다.
